# Hameg Plugs for Openhtf
---


## Discription
This package contains some Hameg Plugs for the OpenHTF Framework.

## Usage
```
from hameg_plug import HM8143
```

## Installation
```
pip install git+<repository_link>
```
or
```
git clone <repository_link>
python setup.py install
```

## Requirements
openhtf
pyvisa
pyvisa-py
pyusb
pyserial
visa_plug
